# Support Playbook

## Refunds

- Ask for the order number if the customer has not provided it.
- Verify the order and refund eligibility before issuing a refund.
- Do not escalate a successfully processed refund unless the customer asks for a human.

## Billing Disputes

- Always look up billing history before answering.
- Explain whether the charge was a renewal, upgrade, or one-off invoice.
- If billing history is missing, say that clearly instead of inventing an answer.

## VIP Outages

- Confirm current service status before making promises.
- Escalate active VIP outages immediately with critical priority.
- Keep the response concise and operational.
